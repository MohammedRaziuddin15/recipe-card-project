# 🍽️ Interactive Recipe Card

An interactive Recipe Card web application built using HTML, CSS, and JavaScript.

## 📌 Project Overview

This project displays a recipe in a clean and responsive card layout. Users can view ingredients, cooking steps, track cooking progress, and see a completion message when all steps are finished.

## 🚀 Features

- Recipe title and image
- Preparation time and serving size
- Show/Hide Ingredients button
- Show/Hide Cooking Steps button
- Start Cooking functionality
- Next Step navigation
- Active step highlighting
- Progress bar tracking
- Recipe completion message
- Responsive design for mobile devices
- Hover effects and smooth transitions

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
